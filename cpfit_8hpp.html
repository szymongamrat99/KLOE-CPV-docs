int cp_fit_mc_data (TChain &chain, TString mode, bool check_corr, Controls::DataType &data_type, ErrorHandling::ErrorLogs &logger, KLOE::pm00 &Obj)
 

Detailed Description

Header file for CP Fit functions with plots generation.

Parameters
chainAddress to the TChain used to get the data
modeMode of operation ("split" by default)
check_corrFlag, if the efficiency correction should be used
data_typeParameter, which type of data should be taken into account:
  • 1: MC signal only
  • 2: MC signal + background
  • 3: MC + Data
  • 4: Entire MC signal (before cuts and with all events with errors)
loggerAddress to the logger for error and info.
ObjAddress to the object from KLOE::pm00 class
Returns
Integer flag:
  • 0: no error
  • 1: error
