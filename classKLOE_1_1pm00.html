Float_t *  trc 
)

Reconstruction of neutral vertex using the triangle method.

Parameters
TrcSumFinalpointer to variable to get the TrcSum after the method application

\[ TrcSumFinal = \sum^{4}_{i=1} (T_{cl,i} - \frac{d_{\gamma,i}}{c} - \frac{d_{K}}{c\beta_{K}}) \]

vtxSigmaFinalpointer to the variable to get the error of reconstructed vertex

\[ vtxSigmaFinal = \frac{\sum^{4}_{i=1} E_{cl,i} \sqrt{(X_{neu,i} - X_{neu,avg})^2 + (Y_{neu,i} - Y_{neu,avg})^2 + (Z_{neu,i} - Z_{neu,avg})^2}}{\sum^{4}_{i=1} E_{cl,i}} \]

Returns
The function returns:
  • 0: no errors
  • ErrorHandling:ErrorCodes::FILE_NOT_EXIST: input file does not exist
